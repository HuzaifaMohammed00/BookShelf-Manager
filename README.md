BookShelf Manager is a command-line inventory system 
built with Python and SQLite. It helps bookstore owners 
manage their library in a professional and organized way.

Features:
- Add new books to the inventory
- Search books by title or author
- View all books (full or simple view)
- Update book quantity and price
- Sell books with automatic stock tracking
- Generate inventory reports

Technologies Used:
- Python 3
- SQLite3 (database)

What I Learned:
- Database design with relationships (Foreign Keys)
- CRUD operations with SQL
- Organizing code into functions and modules
- Building a real-world command-line application

This project was built as part of my Python for Everybody 
specialization on Coursera.
